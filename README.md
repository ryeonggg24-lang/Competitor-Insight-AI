# ☕ Competitor Insight AI

AI 기반 경쟁사 신제품 분석 웹서비스

---

## 프로젝트 목표

상품전략팀에서 사용하는 경쟁사 신제품 분석 업무를
AI를 활용하여 자동화하고,
시장 트렌드 및 시사점을 빠르게 도출하는 웹서비스 개발

---

## 주요 기능 (예정)

- 경쟁사 게시글 분석
- 제품명 자동 추출
- 가격 추출
- 카테고리 분류
- GPT 기반 시사점 생성
- 월간 트렌드 분석
- PPT용 문구 생성
- 엑셀 저장
