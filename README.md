# ☕ Competitor Insight AI

AI 기반 경쟁사 신제품 분석 웹서비스

---

## 📌 Why?

상품기획자는 경쟁사의 신제품을 단순히 수집하는 것이 아니라,
출시 전략과 시장 흐름을 분석하여 자사 상품 기획에 반영해야 합니다.

그러나 실제 업무에서는 정보 수집과 정리에 많은 시간이 소요되어
정작 중요한 분석과 인사이트 도출에 충분한 시간을 확보하기 어렵습니다.

Competitor Insight AI는 경쟁사 데이터를 구조화하고
AI 기반 분석을 통해 상품기획 의사결정을 지원하기 위해 기획되었습니다.

---

## 🎯 프로젝트 목표

Competitor Insight AI는 단순한 경쟁사 신제품 정보 정리 도구가 아니라,
브랜드별 신제품 출시 데이터를 구조화하고 AI 분석을 결합하여
시장 트렌드, 가격 전략, 카테고리 변화 및 상품기획 인사이트를
도출하는 AI 기반 경쟁사 분석 서비스입니다.

본 프로젝트는 상품전략팀 인턴으로 근무하며 수행했던 경쟁사
신제품 모니터링 업무에서 출발했으며,
단순 정보 수집을 넘어 의사결정에 활용 가능한 인사이트를
빠르게 도출하는 것을 목표로 합니다.

### 기대 효과

반복적인 정보 정리 시간을 줄이고,
경쟁사 전략과 시장 트렌드를 보다 빠르고 객관적으로 분석하여
상품기획 의사결정을 지원합니다.

---

## 🚀 주요 기능

### 1. 경쟁사 신제품 분석

- 브랜드별 출시 전략 분석
- 가격 전략 비교
- 시즌별 · 카테고리별 트렌드 분석
- 경쟁사 차별화 포인트 분석

### 2. AI 기반 상품기획 인사이트

- 소비자 타깃 추정
- 핵심 원재료 및 키워드 분석
- 상품기획 시사점 생성
- PPT 보고용 요약 문구 자동 생성

### 3. 자사 및 경쟁사 신제품 통합 관리

- 브랜드
- 출시 연도
- 출시일
- 제품명
- 가격
- 사이즈
- 카테고리
- 주요 특징
- 비고

### 4. 트렌드 대시보드

- 월별 신제품 출시 현황
- 브랜드별 출시 비중
- 카테고리별 변화
- 가격 분포 분석
- 키워드 및 원재료 트렌드 분석
