# ☕ Competitor Insight AI

AI 기반 경쟁사 신제품 분석 웹서비스

---

## 프로젝트 목표

Competitor Insight AI는 단순한 경쟁사 신제품 정보 정리 도구가 아니라,
브랜드별 신제품 출시 데이터를 구조화하고 AI 분석을 결합하여
시장 트렌드, 가격 전략, 카테고리 변화 및 상품기획 인사이트를
도출하는 AI 기반 경쟁사 분석 서비스입니다.

본 프로젝트는 상품전략팀 인턴으로 근무하며 수행했던 경쟁사
신제품 모니터링 업무에서 출발했으며, 단순 정보 수집을 넘어
의사결정에 활용 가능한 인사이트를 빠르게 도출하는 것을 목표로 합니다.

---

## 주요 기능

### 1. 신제품 데이터 구조화
- 브랜드, 연도, 출시일, 제품명, 가격, 사이즈, 카테고리, 특징 정리
- PPT/Excel 보고 양식에 맞춘 표준 데이터 생성

### 2. 경쟁사 전략 분석
- 브랜드별 출시 방향 분석
- 가격대 및 사이즈 전략 비교
- 시즌/원재료/카테고리 키워드 분석
- 전년·전월 대비 출시 흐름 비교

### 3. AI 기반 상품전략 시사점 도출
- 경쟁사 출시 의도 추정
- 소비자 타깃 및 소구 포인트 분석
- 이디야 적용 가능 인사이트 생성
- PPT용 한 줄 요약 및 보고 문구 생성

### 4. 트렌드 대시보드
- 월별 출시 수
- 브랜드별 카테고리 비중
- 주요 키워드 TOP N
- 가격대 변화
- 음료/푸드/MD 출시 비중
