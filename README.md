# ☕ Competitor Insight AI

AI 기반 경쟁사 신제품 분석 웹서비스

---

## 📌 Why?

상품기획자는 경쟁사 신제품을 단순히 수집하는 것이 아니라,
출시 전략과 시장 흐름을 분석하여 자사 상품 개발에 반영해야 합니다.

하지만 실제 업무에서는 경쟁사 정보를 여러 채널에서 직접 확인하고,
PPT 및 Excel 형식으로 수작업 정리하는 과정에 많은 시간이 소요되며,
분석과 인사이트 도출까지 이어지는 과정 역시 대부분 수작업으로 이루어집니다.

**Competitor Insight AI**는 이러한 문제를 해결하기 위해 기획된
AI 기반 경쟁사 분석 서비스입니다.

---

## 🎯 프로젝트 목표

Competitor Insight AI는 단순한 경쟁사 신제품 정보 정리 도구가 아니라,
브랜드별 신제품 출시 데이터를 구조화하고 AI 분석을 결합하여
시장 트렌드, 가격 전략, 카테고리 변화 및 상품기획 인사이트를
도출하는 AI 기반 경쟁사 분석 서비스입니다.

본 프로젝트는 상품전략팀 인턴으로 근무하며 수행했던 경쟁사
신제품 모니터링 업무에서 출발했으며,
단순 정보 수집을 넘어 의사결정에 활용 가능한 인사이트를
빠르게 도출하는 것을 목표로 합니다.

---

## 🚀 주요 기능

### 1. 경쟁사 신제품 분석

- 브랜드별 출시 전략 분석
- 가격 전략 비교
- 시즌별 · 카테고리별 트렌드 분석
- 경쟁사 차별화 포인트 분석

### 2. AI 기반 상품기획 인사이트

- 소비자 타깃 추정
- 핵심 원재료 및 키워드 분석
- 상품기획 시사점 생성
- PPT 보고용 요약 문구 자동 생성

### 3. 신제품 데이터 구조화

- 브랜드
- 출시 연도
- 출시일
- 제품명
- 가격
- 사이즈
- 카테고리
- 주요 특징
- 비고

### 4. 트렌드 대시보드

- 월별 신제품 출시 현황
- 브랜드별 출시 비중
- 카테고리별 변화
- 가격 분포 분석
- 키워드 및 원재료 트렌드 분석
